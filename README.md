## Getting Started

This is a hardhat project. To install required node.js modules:

```bash
npm ci
```

To compile the solidity source code

```bash
npm run compile
```

To run unit test:

```bash
npm run test:normal
```

To deploy the smart contract on Ethereum ropsten testnet:

```bash
npm run deploy:ropsten
```
