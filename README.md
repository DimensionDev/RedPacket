## Getting Started

This is a standard truffle project.
To install:
```
npm i
```
To build the project:
```
truffle build
```

To test the project:
```
npm install chai ganache-cli
ganache-cli -a 100
truffle test
// or test with code coverage
truffle run coverage
```

To debug:
```
truffle debug [TX_ID]
```
